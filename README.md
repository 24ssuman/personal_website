# Suman Saha Personal Website

Static academic website for Suman Saha, Ph.D. Candidate in Chemistry at UC San Diego.

## Files

- `index.html` contains the site structure and public CV-derived content.
- `style.css` contains the visual system, layout, and responsive behavior.
- `assets/` contains local website visuals.

## Privacy

`suman_CV.pdf` is used only as a local reference and is intentionally ignored by git.
